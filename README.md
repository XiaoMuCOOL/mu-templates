## Usage

```console
    # Npm install(安装依赖)
    $ cd project && npm i

    # Run the project(运行项目)
    $ npm run server

    # Package items(打包项目-在原目录下生成.min文件)
    $ npm run build

    # Clean items(清除.min文件)
    $ npm run clean

    # Lint js/css(js/css检查)
    $ npm run lintAll
```
